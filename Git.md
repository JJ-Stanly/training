# Git commands

* Initialise
  ```git
    git init
  ```

* Adding to tracker
  ```git
    git add <file_name> 

        OR

    git add .
  ```

* Commit changes
  ```git
    git commit -m 'message'
  ```

* Connecting to remote (adding remote URL)
  ```git
    git remote add origin <url>
  ```

* push changes from local to remote
  ```git
    git push -u origin 
  ```

* cloning a project
  ```git
    git clone <url>
  ```
